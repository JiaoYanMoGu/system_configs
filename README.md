### 4. 好用的工具搜集
+ `typora`:Markdown编辑器
+ `ack-grep`:代码搜索
+ `tig`:zsh插件,用于查看git log
+ `git`:zsh插件,提供了一堆alias
+ `git-extras`:git 额外命令 git summary等
+ `ydict`:终端内部使用的有道词典
+ `math-pix`:图片生成LaTex公式
+ `pdftk`: pdf文件处理工具:切割合并等
+ `navi`: 命令行备忘录功能，用于常用命令速查[url](https://github.com/denisidoro/navi)
